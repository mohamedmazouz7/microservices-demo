# Microservices Demo - DevOps & Kubernetes Learning Project

[![Kubernetes](https://img.shields.io/badge/Kubernetes-1.27%2B-blue)](https://kubernetes.io/)
[![Docker](https://img.shields.io/badge/Docker-Latest-2496ED)](https://www.docker.com/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

A comprehensive microservices application built for learning DevOps, Kubernetes, and cloud-native concepts. This project demonstrates best practices for containerization, orchestration, CI/CD, and infrastructure as code.

## 🎯 Learning Objectives

This project covers essential DevOps concepts:
- ✅ **Containerization**: Docker, multi-stage builds, security best practices
- ✅ **Kubernetes Orchestration**: Deployments, Services, ConfigMaps, Secrets, StatefulSets
- ✅ **Networking & Service Mesh**: Ingress, Service-to-service communication, gRPC
- ✅ **Configuration Management**: Helm charts, Kustomize overlays
- ✅ **CI/CD Pipelines**: GitHub Actions, automated testing & deployment
- ✅ **Infrastructure as Code**: Terraform, cloud resource management
- ✅ **Monitoring & Logging**: Application metrics and observability
- ✅ **Security**: RBAC, Pod Security Policies, image scanning

## 🏗️ Project Architecture

**Online Boutique** is an 11-microservices e-commerce application demonstrating cloud-native patterns.
